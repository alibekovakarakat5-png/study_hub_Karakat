## 1. Оффер (что говорим директору)

**«Пилот к сентябрьскому набору»** — только для первых 2–3 центров:

- 4 недели полного доступа на тарифе «Центр» (до 20 учителей, 200 учеников) — бесплатно;
- загружаем ВАШИ материалы (PDF/DOCX) → AI режет на уроки → ученики получают
  персональные домашки из ваших материалов;
- обучение учителей — 1 час онлайн, онбординг «5 шагов» (ниже);
- приоритетная поддержка лично от основателя весь пилот.

Взамен центр даёт:
- видео-отзыв директора по итогам (2–3 минуты);
- право на кейс с анонимизированными цифрами («центр N в Астане: +X% выполнения ДЗ»).

После пилота: тариф «Центр» — 50 000 ₸/мес по счёту на ТОО (юрлицам привычно,
карта не нужна). Если не подошло — просто расстаёмся, данные центра выгружаем.

**Дедлайн-рычаг (честный):** набор учеников — сентябрь. Чтобы учителя освоились
и материалы были загружены к 1 сентября, пилот должен стартовать до 25 августа.

## 2. Договор пилота — 1 страница (структура)

1. Стороны: ТОО «___» (StudyHub) и центр «___».
2. Срок: 4 недели с даты старта, без автоматического перехода в платный.
3. StudyHub предоставляет: доступ «Центр», загрузку материалов, обучение 1 час,
   поддержку в WhatsApp в рабочие часы.
4. Центр предоставляет: до 3 учебных материалов (PDF/DOCX), 1 контактного
   администратора, видео-отзыв и согласие на кейс по итогам.
5. Данные: материалы центра видны только центру (multi-tenant), по окончании —
   выгрузка или удаление по запросу.
6. Стоимость пилота: 0 ₸. После пилота — по желанию центра, тариф «Центр»
   50 000 ₸/мес, оплата по счёту.

## 4. Онбординг центра — «5 шагов до первого ДЗ»

1. **День 1:** создаём организацию, директор получает логин; центр присылает
   1–3 PDF своих материалов.
2. **День 1–2:** загружаем материалы → AI генерирует уроки → проверяем разметку тем.
3. **День 2:** час обучения учителей (создать класс, назначить ДЗ, читать аналитику).
4. **День 3:** учителя заводят классы, ученики получают логины (CSV-импорт).
5. **День 3–7:** первое персональное ДЗ каждому ученику; через неделю — разбор
   дашборда с директором (первые цифры активности).

## 5. Метрики пилота (что показываем директору в конце)

- % учеников, заходивших ≥3 раз в неделю;
- % выполнения домашек (до/после — если центр даст базу сравнения);
- топ слабых тем по группам (чего не видно без платформы);
- количество вопросов к AI вне уроков (= вовлечённость между занятиями).

Эти же 4 цифры — скелет публичного кейса для продажи следующих центров.
